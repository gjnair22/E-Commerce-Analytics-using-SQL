# 🛒 E-Commerce Sales Analytics using SQL (MySQL)

##  Project Overview

This project analyzes a real-world e-commerce dataset using **MySQL** to extract business insights related to customers, orders, products, payments, and sales performance.

The objective is to simulate the role of a Data Analyst by answering business-driven questions using SQL, helping stakeholders understand customer behavior, product performance, payment trends, and overall business performance.

---

##  Objectives

- Analyze customer purchasing behavior
- Measure sales and order performance
- Understand product and category trends
- Evaluate payment preferences
- Generate business insights using SQL
- Practice real-world SQL concepts used in data analytics

---

##  Dataset

**Dataset:** Brazilian E-Commerce Public Dataset by Olist

The dataset contains information about:

- Customers
- Orders
- Products
- Order Items
- Payments
- Reviews
- Product Categories
  
Due to GitHub file size limitations, the complete dataset is not included in this repository.

Dataset Source:
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce
